# Mesh networks simulation


## Стабильная работа

- Операционная система: Ubuntu 20.04 (рекомендуется) 
- OMNeT++ версии **5.7.1**
- Библиотека **INETMANET 3.8.2** (устанавливается через `opp_env`, см. ниже)

## Установка
1. Рекомендуется устанавливать OMNeT++ и INETMANET 3.8.2 с помощью менеджера окружений opp_env
2. Клонируйте проект в ../omnetpp-5.7.1/samples 
3. Добавьте INETMANET в окружение проекта через project-references
